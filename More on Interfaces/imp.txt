We have to use super keyword only for using default methods of an interface.
Ex : 

Interface A {
    default ...
}

public class CLS {
    psvm {
        A.super.MethodName();
        i.e., InterfaceName.super.MethodName();
    }
}

But if the method is static...
Ex : 

Interface A {
    static...
}

public class CLS {
    psvm {
        A objName = new CLS();
        i.e., InterfaceName objName = new ClassName();
    }
}
